# 🎯 K-Nearest Neighbors (KNN) Classification Project

This project demonstrates the implementation of the **K-Nearest Neighbors (KNN)** Machine Learning algorithm to predict customer purchasing behavior using the **Social Network Ads** dataset.

## 📌 Project Overview

The objective of this project is to classify whether a customer will purchase a product based on features such as **Age** and **Estimated Salary**. The KNN algorithm is trained and evaluated using Python and Scikit-learn.

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Jupyter Notebook
